4.COFFEE SALES ANALYSIS WITH EXCEL

Academic Projects

PREDICTING CARDIOVASCULAR DISEASE BY USING MACHINE LEARNING
In this study, a machine learning model was developed using the Cleveland heart disease dataset to predict cardiovascular disease. Utilizing three algorithms (Decision Tree, KNN, and ANN) and addressing data issues through preprocessing and feature selection, the study found that the KNN model outperformed the others after feature selection. The research highlights the effectiveness of machine learning in predicting cardiovascular illness and underscores the importance of high-quality datasets and tailored feature selection for optimal model performance.
